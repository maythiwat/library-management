# Library Management System

This project is a part of my Ruby on Rails Bootcamp for learning how to CRUD in Ruby on Rails.

## Project overview
- User Profile
- Book Catalog
- Book Loan
- Roles (Admin, Librarian, User)
- Book Audit Logs
- Dashboard (for Admin and Librarian)
- Vibe Coded UI
